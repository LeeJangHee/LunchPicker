# Launch Picker
---

- 사용 설명서
    1. 흔들기 버튼을 클릭한다.
    2. "흔들어 주세요~" 텍스트가 보이면 휴대폰을 흔든다.
    3. 텍스트가 변경되면서 결정된다.
    4. 마음에 들지 않으면 "다시 고르기"를 통해 다시 고른다.
    5. 우측 상단 버튼을 누르면 먹기 싫은 메뉴를 고를 수 있다.
